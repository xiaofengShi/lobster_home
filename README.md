# 🦞 龙虾管家 (Lobster Home)

> 基于 OpenClaw 的智能家居 AI 管家 — 不只是自动化，而是真正理解你的家

## ✨ 核心能力

### 🔊 双向语音交互
对小爱音箱说"**小爱同学，龙虾明天天气怎么样**"，龙虾管家自动接管处理。

**双人格共存**：
- 🤖 "小爱同学" → 原版小爱（开灯、放歌、查快递）
- 🦞 "龙虾" → AI 管家（天气、日程、巡检、智能对话）

技术：轮询小米云端对话 API → 唤醒词过滤 → AI 处理 → 音箱 TTS 播报

### 📷 主动看护巡查
- VLM（235B 视觉大模型）分析摄像头画面
- **认人**：根据外貌特征识别家庭成员（不只是"检测到1人"）
- **天气联动**：自动查询天气，下雨提醒关窗、大风提醒收衣服
- **异常检测**：凌晨运动报警、温度/湿度异常提醒、门长时间未关
- 隐私优先：20:00 后停止摄像头，用户关闭的设备绝不自动打开

### 🌅 早间智能播报
不是固定时间闹钟，而是**摄像头检测到你起床活动后才播报**：
- 今日天气 + 降雨/大风预警
- 待办提醒 + 课程安排
- 在家时音箱播报，不在家时飞书推送

### 📋 口述提醒系统
> "明天下午5点有个热力模型周会"

说一句就记住，提前 15 分钟飞书 + 音箱提醒。支持单次/每日/每周重复。

### 🧠 行为学习
- `lobster_brain.py`：观察 → 学习 → 记忆
- 越用越准的人物识别
- 自动发现家庭作息规律

## 🎯 设计理念

**不是"如果X则Y"的自动化规则，而是理解上下文的 AI 决策。**

- 📍 **场景感知**：在家 → 音箱播报；不在家 → 飞书推送
- 🕐 **时间感知**：早上播天气，下午提醒接小宝，晚上不打扰
- 👤 **人物感知**：识别家庭成员，个性化提醒
- 🌤️ **环境感知**：天气变化联动提醒
- 🔒 **隐私优先**：用户关闭的设备绝不打开，20:00后停止摄像头
