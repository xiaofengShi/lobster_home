# 🦞 Lobster Home

**会进化的家庭 AI——不是智能家居，是一个真正在思考的家庭成员。**

👉 [在线体验](https://xiaofengshi.com/lobster_home/) · [真实运行 Demo](https://qcnht9ffvtkv.feishu.cn/wiki/RdJQwhu50iwUoUkJyYPclDwfnvc)

## 它是什么

Lobster Home 是一个基于 [OpenClaw](https://openclaw.ai) 的家庭 AI 系统。它不写 if-else 规则，而是通过**感知→理解→行动→学习**的闭环，自主管理家庭日常。

## 解决什么问题

传统智能家居需要你预先设定每一条规则。Lobster Home 不需要——它自己观察你的生活习惯，自己判断该做什么，做完之后从反馈中学习。

| 传统方式 | Lobster Home |
|---------|-------------|
| 手动写自动化规则 | AI 自己观察、自己判断 |
| 规则写死，不会变 | 每天都在进化，越用越懂你 |
| 只能执行预设动作 | 会自己写代码实现新功能 |
| 被动等指令 | 主动感知、主动行动 |

## 能干什么

**🌅 晨间自动播报** — 起床后自动播天气、通勤、日程，不用问

**🧠 AI 日报 → 语音播客** — 每天采集 AI 领域最新动态，看你在不在家决定用音箱读还是发飞书

**📬 口述提醒** — 说一句"明天下午3点周会"，到点自动飞书+音箱双通道提醒

**🏠 主动看护** — 摄像头+门锁+温湿度交叉验证，判断谁在家、家里什么状态

**🌙 晚间收尾** — 检查门锁、灯光、窗户，该关的关，该提醒的提醒

**🌦️ 天气预警** — 即将下雨自动提醒收衣服关窗户

## 实际运行数据

- **166** 个已注册实体（灯/锁/传感器/音箱...）
- **17** 个自动化场景在跑
- **16** 天持续运行
- **4** 个隐私硬规则（摄像头 20:00 后停巡、用户关的设备绝不自己开）

## 自我进化能力

这不是一个固定功能的产品——它会自己成长：

1. **观察习惯** → 发现你每天早上问天气
2. **产生灵感** → 自己接入气象 API 写好播报逻辑
3. **自主实现** → 此后每天起床自动播报，无人教它这样做
4. **从反馈迭代** → 你说"太啰嗦了"，它自己精简格式

## 隐私设计

- 🔴 摄像头 20:00 后停止巡查（硬编码，不可覆盖）
- 🔴 用户关闭的设备，AI 绝不自行开启
- 🔴 被动传感器优先，减少主动干预
- 🔴 临时数据用完即焚

## 技术栈

- **AI 引擎**: [OpenClaw](https://openclaw.ai)
- **智能家居**: Home Assistant + 米家
- **通知**: 飞书 API + 小爱音箱 TTS
- **语音**: Edge-TTS (中文) + Opus 编码
- **部署**: macOS + launchd 定时任务
